## Stack

- Framework: Next.js
- Authentication: NextAuth
- Database: MySQL (Prisma ORM)
- Logging: Winston (file-based)
- Testing: Jest + Supertest
- Theme: next-themes (Light / Dark / System)
- i18n: Custom dictionary-based (en, th); cookie + localStorage
