# MVP Boundary

**Updated:** mvp-boundary (24/02/2026)

**Source:** PRD §13

---

## MVP Must Have

- Authentication (Email + Google)
- Activity Log for critical events
- System Log at error level

## MVP Out of Scope

- Organization / Project / Task management (to be rebuilt as needed)
- Advanced analytics
- Real-time notifications
- Third-party integrations

**Note:** The current implementation also includes **Income & Expense** (transactions, calendar, Data Tools for CSV import/export), documented in PRD §18; these are not part of MVP must-have.
